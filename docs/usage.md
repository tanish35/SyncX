# Usage

## Connect

1. Sign in with Stremio or paste a Stremio auth key.
2. Connect Nuvio from the dashboard.
3. Use `Sync Now` to pull and push watch progress between both services.

## Inspect Fetched Data

Use `Pull only` on the dashboard to fetch from connected providers without pushing changes back.

Open `/currently-watching` to inspect stored movies, series, episodes, timestamps, and progress. Series are grouped and expandable. Long lists load as you scroll.

## Settings

Use Settings to toggle notifications or disconnect providers.
